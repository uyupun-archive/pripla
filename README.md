# Pripla

<img src="./front/assets/images/logo/logo_black.png" width="200px">

**Princess & Prince date planning service.**

### コマンド

```bash
# 環境構築
$ make setup
# 開発用サーバとコンテナの起動
# フロントエンド: localhost:3000
# バックエンド: localhost:8000
$ make up
# コンテナの終了
$ make down
# Dockerのプロセス確認
$ make ps
# DBにテーブル作成 + シーダーの実行
$ make db
# Linterによるコード修正
$ make lint
```
